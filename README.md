# Airbnb DAV Project

This project analyzes Airbnb pricing and demand patterns for New York and Chicago.

## Project Structure

- `data/raw/`: raw city datasets
- `data/processed/`: cleaned and analysis-ready outputs
- `notebooks/`: step-wise analysis notebooks
- `src/`: reusable Python modules
- `outputs/`: generated plots and tables
- `presentation/`: slide assets
